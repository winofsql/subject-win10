## キーボード ショートカット

| キー | 内容 ♥
| :--- | :--- 
| CTRL + SHIFT + DEL | キャッシュ削除のダイアログ表示( Firefox、Edge も同じ )
| CTRL + SHIFT + B | ブックマークバーの表示・非表示の切り替え
| CTRL + SHIFT + T | 閉じたタブを開き直す
| F12 | デベロッパーツール
| CTRL + プラスキー | 拡大
| CTRL + マイナスキー | 縮小
| CTRL + 0 | 100% サイズに戻す
| CTRL + SHIFT + N | シークレットウインドウを開く
| CTRL + N | 通常ウインドウを開く
| CTRL + H | 履歴
| CTRL + U | 現在表示しているページのソースを表示
| CTRL + S | 現在表示しているページを名前をつけて保存
| CTRL + T | 新しいタブ
